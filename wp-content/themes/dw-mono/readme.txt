== Description ==
DW Mono theme features are clean, lightweight designs, all with a minimal file size. This theme will bring you an exciting experience with these following features:

* Simple yet Unique Design
* Mobile-first, Responsive Layout
* Lightweight Code Base
* Custom Header
* Social Links

For more information about DW Mono please go to https://www.designwall.com/wordpress/themes/dw-mono.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in DW Mono in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
